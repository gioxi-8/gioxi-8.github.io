GioXi-8

GioXi-8 is a custom 8-bit computer based on the Zilog Z80 CPU with a Raspberry Pi Pico acting as a modern video, audio and storage co-processor.

System Overview

  • CPU: Z80 @ 8 MHz
  • RAM: 64 KB static
  • ROM: 16 KB (monitor + Forth)
  • Video/Audio/SD handled by RP2040
  • Output: composite PAL or VGA
  • Audio: PWM stereo or I²S DAC
  • Storage: microSD (SPI)

Project Status

Currently working on the hardware design in KiCad and finalizing the memory/bus architecture. The goal is a modular 8-bit system inspired by classic home computers but enhanced with modern peripherals.

Next Steps

  1. Finish schematic and PCB layout.
  2. Define ROM structure and Forth kernel.
  3. Develop Pico firmware (video/audio).
  4. Assemble prototype and run first boot.
